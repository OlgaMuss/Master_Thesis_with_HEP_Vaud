## NOTES POUR QUALTRICS IMPORT

### Types de Questions Qualtrics:
- **Text** → Text Entry (Single Line)
- **Long_Text** → Text Entry (Essay Box, minimum 100 characters)
- **Single_Choice** → Multiple Choice (Single Answer)
- **Multiple_Response** → Multiple Choice (Multiple Answers)
- **Likert_5** → Matrix Table or Slider (1-5 scale)

### Logic/Branching:
- Display Q3_1_cognitive_other only if "Autre" selected in Q3_1_cognitive_elements
- Similar conditional display for all "_other" questions

### Validation Rules:
- Long_Text questions: Minimum 50-100 characters
- Likert scales: Force response
- Multiple response: Minimum 1 selection

### Page Breaks:
- After Section 1 (Demographics)
- After Section 2 (Part 1 - Individual Reflection)
- After Section 3 (Part 2A - Monitoring)
- After Section 4 (Part 2B - Pedagogical Gestures)
